# Kamstrup Omnipower HAN module with ESP8266

HAN module for Kamstrup Omnipower based on ESP8266

The design is done with EasyEDA and PCB produced by JLCPCB (https://jlcpcb.com/). 10 pcs 2 layer is ~4US$ !
